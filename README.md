## FIRST SETUP
- Run npm:
  ```md
  npm i
  ```
- Create file .env have properties:

    * PORT
    * URL_DATABASE

## RUN APP FOR DEV
- Create first new terminal and run:
  ```md
  npm run start:dev
  ```

- Create second new terminal and run:
  ```md
  npm run ts:watch
  ```

- Wait app log content:
  ```md
    App listening on port 5000
    Connect db success
  ```

- You can test any API
